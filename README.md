# 🛒 Amazon Product Review Sentiment & Trend Analysis

An AI-powered Machine Learning application that analyzes Amazon product reviews and predicts whether the sentiment is **Positive** or **Negative** using Natural Language Processing (NLP). The application also provides interactive visualizations and trend analysis through a Streamlit dashboard.

---

## 🚀 Features

- 🤖 Sentiment Analysis using Machine Learning
- ⭐ Positive/Negative Review Prediction
- 📊 Interactive Dashboard
- 📈 Trend Analysis
- 🎯 Confidence Score
- 📂 Bulk CSV Analysis
- 📝 Prediction History
- 💾 Download Report
- 🎨 Professional Streamlit UI

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-Learn
- Pandas
- Plotly
- Joblib

---

**Amazon Fashion Reviews Dataset**
